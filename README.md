# 木序咖啡 MUXU COFFEE

台北赤峰街自家烘焙咖啡館形象官網 — 多頁式靜態網站。

- **首頁** `index.html`：品牌主張、招牌品項、從豆到杯流程、客戶評價
- **咖啡與菜單** `products.html`：手沖／義式／季節飲品、自家烘焙豆價目、佐咖啡甜點
- **品牌故事** `about.html`：品牌起源、烘焙理念、職人團隊
- **門市・預約** `contact.html`：門市資訊、營業時間、地圖、線上預約表單

## 技術
- 純靜態 HTML / CSS / JS，無建置流程
- 共用導覽列與頁尾由 `js/shared.js` 注入
- 響應式設計（RWD），繁體中文內容
- 所有情境圖片以 Higgsfield AI 生成，Logo 為自訂 SVG
- 部署於 GitHub Pages
